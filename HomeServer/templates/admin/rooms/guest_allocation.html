Quick Guide
Process:
  1. Create walk-in guest (if new) via "Identity & Access → Guests"
  2. Select vacant room from dropdown
  3. Select guest from dropdown
  4. Set expected checkout (optional)
  5. Click "Check-in Guest"
What happens:
  • Guest status becomes CHECKED_IN
  • Room status becomes ACTIVE
  • Room is marked as occupied
  • Device permissions are applied
Timezone: All times are in Kampala time (UTC+3)
